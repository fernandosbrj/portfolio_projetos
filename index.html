Projetos em Ciência de Dados

Avaliação de desempenho escolar utilizando modelos lineares hierárquicos multivariados.

Utilizei liguagem R para limpeza, preparação dos dados e modelagem univariada, e SAS para modelagem multivariada. O problema avaliou quais fatores são mais importantes para explicar o desempenho escolar dos alunos utilizando variáveis tanto do aluno quanto da escola. Para isso, foi utilizada a técnica de modelos hierárquicos, mais adequada para trabalhar com dados e variáveis de grupo (alunos agrupados em escolas, por ex).

As ferramentas utilizadas foram:

  • Linguagem R, IDE R-Studio
  • SAS

Previsão de demanda por energia elétrica residencial no Brasil

Utilizando modelos e métodos de séries temporais, fiz a comparação da performance do modelo probabilístico SARIMA e o método Holt-Winters na previsão de demanda por energia elétrica residencial no Brasil com dados de 2013 a 2020.

As ferramentas utilizadas foram:

  • Linguagem R, IDE R-Studio

Avaliação dos fatores determinantes na média de pontos por jogo de uma equipe basquete utilizando o paradigma bayesiano

O estudo de caso propôs avaliar os fatores determinantes na média de pontos por jogo de uma equipe de basquete participante do Novo Basquete Brasil através de um modelo linear utilizando o paradigma bayesiano para estimação dos coeficientes e intervalos de credibilidade.

As ferramentas utilizadas foram:

  • Linguagem R, IDE R-Studio

Avaliação do custo de contratos para construção de poços submarinos

Avaliação dos fatores determinantes e estimação do custo de contratação para exploração de poços submarinos em Reais e Dólares. O problema envolveu a comparação de diferentes modelos de regressão (Regressão linear, Random Forest e SVR) para estimação do custo em questão

As ferramentas utilizadas foram:

  • Linguagem Python, Jupyter Notebook, Gitlab
